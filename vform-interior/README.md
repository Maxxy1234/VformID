✨ **Modern Luxury Design**
- Premium color palette with gold accents

## How to Use

1. **Extract the files** to your web server or local directory
2. **Replace image placeholders** with actual project photos
3. **Update contact information** in the HTML (WhatsApp number, email, social links)
4. **Customize copy** as needed
5. **Deploy to Vercel** or any static hosting service

## Customization Guide

### Update WhatsApp Number
Find and replace `60123456789` with your actual WhatsApp number throughout the HTML and JS files.

### Replace Image Placeholders
- Portfolio images: Replace `.image-placeholder` divs with actual `<img>` tags
- About section images: Update the two overlapping image boxes
- Maintain aspect ratios for best results

### Update Social Links
In the footer section, update Instagram and Facebook links to your profiles.

### Modify Content
All text content can be easily edited directly in the HTML file.
